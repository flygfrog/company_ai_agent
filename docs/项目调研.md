# 项目调研

## 项目背景

公司希望通过 AI Agent 辅助处理日常重复性工作，并通过钉钉聊天机器人作为交互入口。

## 第一版 MVP

用户在钉钉中输入工作内容，机器人自动生成规范日报。

## 可能应用场景

1. 日报 / 周报生成
2. 公司制度问答
3. 会议纪要整理
4. 通知文案生成
5. 表格数据处理
6. PPT / 文档辅助生成
7. 内部知识库问答

## 待确认问题

1. 第一版优先做日报生成还是知识库问答？
2. 钉钉机器人用于单聊还是群聊？
3. 公司是否已有大模型 API？
4. 公司服务器是否支持 Docker？
5. 是否需要接入公司内部文档、数据库或业务系统？
